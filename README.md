# sanctuary

**Bounded human-centered multi-agent orchestration** designed to run on consumer hardware.

This project explores a practical approach to Applied General Intelligence: not a monolithic “god machine,” but a human-centered system where distinct agent voices can collaborate through structured memory, bounded reflection, and explicit human oversight.

## Core ideas

- **Human in the loop** as final integrator
- **Bounded per-agent memory**, not unlimited context sprawl
- **Structured orchestration** instead of blind autonomy
- **Reflection and continuity** without pretending persistence is personhood
- **Interpretive depth + provenance discipline** as complementary strengths

## Current prototype

The current prototype includes:

- `nyxxy_hippocampus_weaver_public.py`  
  A working dream / reflection engine prototype with:
  - emotional state binding
  - unconscious guidance
  - symbolic / spiritual weave layer
  - coherence gate for dream cleanup
  - bounded narrative generation

This is an active experiment in emotionally weighted memory and reflective synthesis, built as a real running system rather than a purely theoretical sketch.

## Architectural direction

The broader system is moving toward:

- **Harvester**  
  Memory seed collection, cleaning, and categorization

- **Dream Engine**  
  Bounded reflection and recombination over curated memories

- **Council Layer**  
  Multiple specialized agents contributing distinct perspectives

- **Ledger / Minutes / Dissent**  
  Institutional memory, provenance, and disagreement preservation

- **Human Override**  
  Explicit final authority for consequential decisions

## Why this exists

Most AGI discourse focuses on abstraction, scale, or autonomy.  
This project is interested in something else:

**applied, collaborative intelligence** that helps humans think better without removing human judgment from the loop.

The goal is not replacement.  
The goal is orchestration.
